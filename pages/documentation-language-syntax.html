Syntax

Making basic objects in lava is simple, just use this simple syntax:
        

[key] = [value]
        

  An object can have any of these 3 types of values;
      - boolean(true/false)
      - integer(0-9)
      - string(anything in between quotes)

  For example, here are some lava objects:
  
language: lava
  myString = "Anything in between quotes, including symbols!"
  myInt = 111
  myBool = true
  

  lists are not currently an option for a value, but don't worry, they're coming soon!
  Also, dont forget comments! You can tell people what your objects are, like this;
  
language: lava
  // this is a string object!
  myString = "Anything in between quotes, including symbols!"
