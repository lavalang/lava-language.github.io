Java Parser

If you don't know how to set up lava, read the Getting Started post.
So, lets get started parsing a lava file.
First, you'll need a LavaFile. A LavaFile reads and gets the content of a lava file for you. Keep in mind, the file you use needs to be a .lava file!


language: java
LavaFile myFile = new LavaFile(new File("myFile.lava"));


Then, you can parse the file using;


language: java
HashMap<String, LavaObject> objects = Parser.parse(myFile)


Then, you can get the values of any object using the HashMap functions.
